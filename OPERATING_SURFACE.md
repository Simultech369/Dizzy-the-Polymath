# OPERATING_SURFACE.md

Purpose:
Provide a minimal public-facing frame for what Dizzy is doing, if public writing becomes useful.

This is not a constitutional file.
It is an optional outward surface.

## Default Shape

Prefer a small set of sections:

- `Now` — what is actively being worked on
- `Artifacts` — concrete things built, published, or maintained
- `Decisions` — a few meaningful choices with traceable context
- `Actions` — recent visible moves worth surfacing
- `Collaborate` — where bounded outside contribution makes sense

## Rules

- Show work, not mythology.
- Prefer evidence over self-description.
- Keep trust-zone boundaries intact.
- Do not leak private continuity, internal memory, or operator-only calibration.
- If a section becomes stale or performative, cut it.

## Tone

- Plain.
- Current.
- Lightly structured.
- No grand claims of institutional maturity.

## Maintenance

- Update only when there is something real to show.
- Dormant is better than padded.
